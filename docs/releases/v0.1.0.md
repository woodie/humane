# humane v0.1.0

## Initial release

`humane` formats file sizes and relative dates the way macOS Finder does, modeled on Swift's `ByteCountFormatter` and `RelativeDateTimeFormatter`: small, configurable formatter types with a single `Format` method, rather than bare functions. Extracted from real usage in `lambada` and `scandalous`, which were carrying hand-rolled, drifting copies of the same logic.

## SizeFormatter

Formats byte counts with 1000-based math and capitalized unit labels ("80 KB", "1.5 MB"), matching Finder's own file-size display — not SI's lowercase "kB", and not the 1024-based math paired with "KB" labels that most humanize libraries (including Rails' `number_to_human_size`) actually use.

## TimeFormatter

Formats one time relative to another as "X ago" / "X from now", with no "about" prefix on the hour bucket. `CollapseMinute` (on by default via `NewTimeFormatter`) buckets anything under a minute as "less than a minute ago" / "less than a minute from now".

This first release is scoped to just these two formatters. See the README for usage and `docs/COMMENTS.md` for design rationale.
